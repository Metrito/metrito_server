- Criando um Dashboard:

1. Primeiro clique na opção "Dashboards" no menu superior, um dropdown será acionado, em seguida clique na opção "Novo Dashboard" e você será levado para a página de criação de Dashboard.

2. Selecione a empresa a qual pertencerá o Dashboard (Nesse caso é a empresa de nome Facebook que criamos previamente)

3. Dê um nome para o Dashboard

4. Dê uma descrição para o Dashboard (este passo é opcional)

5. Selecione um template pronto (Template Metrito no Modal que irá abrir) para utilizar. Para selecionar basta clicar no card referente ao template que deseja utilizar.

6. Clique no botão "Criar Dashboard" e pronto.

7. Para abrir o Dashboard você apenas precisa clicar na opção "Meus Dashboards" novamente no dropdown do menu superior e em seguida, após carregada a lista de seus Dashboards, basta clicar no Dashboard que acabou de criar ou no ícone de 'abrir em nova guia' no canto direito da tabela.

8. Com o dashboard aberto em uma nova guia basta selecionar a Conta de Anúncio e o período e em seguida clicar em "Buscar dados" para popular o Dashboard com as informações.

9. OBS.: Como esta Conta de Anúncios acabou de ser criada no perfil do usuário de testes, e nunca veiculou nenhum anúncio, não haverão dados para popular o Dashboard recém criado, porém o funcionamento pode ser visto no link de exemplo apresentado acima.
